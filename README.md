## Terminal Commands

1. Install NodeJs
2. Open Terminal
3. Go to your file project
4. Run in terminal: ```npm install```
5. Then: ```npm start```
6. Navigate to `http://localhost:3000/` (http://cerialize-data-web-app-jesseroyc.c9users.io:8080/maps/vector-maps)
7. (optional) Run in terminal `npm i -g cross-env`
